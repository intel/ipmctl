ipmctl

ipmctl is a utility for configuring and managing Intel® Optane™ PMem modules.

Find out more at the Github site link.

Packages

Several Linux Distributions have ipmctl packages available.
  • SUSE has ipmctl available. More information at https://build.opensuse.org/package/show/hardware:nvdimm/ipmctl.
  • Red Hat's EPEL repositories contain ipmctl which can be found here https://src.fedoraproject.org/rpms/ipmctl.
  • Ubuntu packages of ipmctl can be found at https://launchpad.net/ubuntu/+source/ipmctl.

Workarounds

When using 2.0.x versions of ipmctl software to update or downgrade firmware on Intel® Optane™ PMem 100 Series modules, please use the “-lpmb” CLI option (use DDRT Large Payload transfer). Otherwise the operation may take significantly longer than it normally would.

Steps reported to have worked for build on SUSE Linux in home directory

Replace homedir with your home directory 

$ git clone https://github.com/intel/ipmctl.git
# zypper in libndctl-devel ruby2.5-rubygem-asciidoctor
$ cd ipmctl
$ mkdir output
$ cd output
$ cmake -DRELEASE=ON -DSAFECLIB_SRC_DOWNLOAD_AND_STATIC_LINK=ON -DCMAKE_INSTALL_PREFIX=/home/homedir/ipmctl/ ..
$ make all
